PowerPointLabs

Providing power user capabilities to the average user



PowerPointLabs is a free add-in for Microsoft PowerPoint, used by over 200,000 people worldwide.

The add-in allows users to fully utilise the capabilities of Microsoft PowerPoint, lowering the learning curve and also introduces new functions to Microsoft Powerpoint. Here are some of the cool features that it has.

For this project, I helped in solving some bugs and creating a new feature (The Positions Lab, which as of this writing, has yet to be released) which provides features that aids users in the positioning of their objects within a slide. This feature was created from scratch by Oswell, another project team member, and myself, under the guidance of our mentor Kai Xie. We spent about 3 months on the project.

To find out more about the add-in, feel free to visit the website here. It's free to use!
